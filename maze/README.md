## 在矩阵中寻找路径总数

有一个m*n的棋盘（矩阵），一个骑士在棋盘的左上角，他要走到棋盘的右下角，一次只能走一步，每次只能向右走一格或者向下走一格。

计算一共有多少条路径可以走。

### 时间复杂度

    O(m*n)

### 空间复杂度

    O(min(m, n))

## 在有障碍的矩阵中寻找路径总数

与[在矩阵中寻找路径总数](#在矩阵中寻找路径总数)类似，只是在矩阵中加入障碍物，
障碍物用1来表示，如下所示：

    [
      [0,0,0],
      [0,1,0],
      [0,0,0],
    ]

以上矩阵的唯一路径总数是2。
